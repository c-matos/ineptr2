# Finalize a chunk by validating and renaming from .part to .json

Finalize a chunk by validating and renaming from .part to .json

## Usage

``` r
finalize_chunk(temp_path, final_path)
```

## Arguments

- temp_path:

  Path to the temporary .part file

- final_path:

  Path to the final .json file

## Value

TRUE on success, FALSE on failure
